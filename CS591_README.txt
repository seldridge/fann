Clone this repo and run the FANN setup options here (http://leenissen.dk/fann/wp/help/installing-fann/)
Go into examples and run make all. If you get an error, try ldconfig.
To run an example of the convolutional neural network, go to examples, run make all, and run ./simple_train_conv
To run an example of how I got my data, run time ./simple_train, and vary the hidden layer size in simple_train.c as desired.
